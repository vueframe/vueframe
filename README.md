# vueframe

### Quick Start

#### Install

```bash
# npm
npm install @vueframe/vueframe
```

#### Import

``main.ts / main.js``

```js
import { createApp } from 'vue'
import App from './App.vue'
import Vueframe from 'vueframe'

const app = createApp(App)
app.use(Vueframe)
app.mount('#app')
```
